# mRestaurant
Aby uruchomic potrzebujesz [Node.js 19.2](https://nodejs.org/en/)

W pobranym repozytorium komenda:
`npm install nodemon -g`
następnie
`npm install`

Aby uruchomić:
uzupełnij wartości w pliku `.env.example` a następnie zmień jego nazwę na `.env`

Następnie:
`npm run dev`
